Collections -> ArrayList

ArrayList forma parte de Collections Framework e implementa la interfaz List y pertenece al package java.util.

ArrayList

ArrayList forma parte de Collections Framework e implementa la interfaz List y pertenece al package java.util.

Es una colección de tipo List de tamaño dinámico a diferencia de un array convencional, es decir un ArrayList puede modificar su tamaño a lo largo de la ejecución del programa y no necesita ser inicializado con un tamaño por defecto.

Inicializar un ArrayList

Un ArrayList se inicializa por defecto con un tamaño de 0 elementos si no indicamos la cantidad de elementos en su construcción. 


      // Inicializar una nueva lista de palabras
      ArrayList<String> palabras = new ArrayList<String>();

Inicializar un ArrayList con un tamaño inicial

Otra forma de crear un ArrayList es indicar su tamaño en el constructor. El siguiente código inicializa un ArrayList de Integer de 4 elementos. 


      // Inicializar una nueva lista de 4 palabras
      ArrayList<String> palabras = new ArrayList<String>(4);
Al indicar la cantidad de elementos iniciales, se crearan todas las posiciones internamente en null

Agregar Elementos a un ArrayList

Usando el método add, recibe como argumento el elemento que se desea agregar a la lista. El elemento se agregará en la siguiente posición disponible. 


    // Inicializar una nueva lista de palabras
    ArrayList<String> palabras = new ArrayList<String>();
    palabras.add( "Hola" );
    palabras.add( "Como estas ?" );
    palabras.add( "Adios :)" );

Agregar un Elemento en una posición determinada

Sobrecarga del método add, indicando la posición (índice comienza en 0) y el elemento que se desea agregar. 


    palabras.add( 2, "Que bien" );

Obtener un Elemento de un ArrayList

Utilizando el método get, se obtiene el elemento según la posición indicada 


    palabras.get( 0 );

Si la posición que se desea acceder esta fuera del rango del array list, el metodo lanzará la excepción "java.lang.IndexOutOfBoundsException" 


    palabras.get( 6 );

Eliminar un Elemento de un ArrayList

Utilizando el método remove, elimina el elemento según la posición indicada. 


    palabras.remove( 0 );

Iterar un ArrayList utilizando un ciclo For

El método size() retorna el tamaño total de la lista. 


for( int i = 0; i < palabras.size() ; i++ ) {
  System.out.println( palabras.get(i) );
}

Iterar un ArrayList utilizando un ciclo ForEach


for( String palabra : palabras ){
   System.out.println( palabra );
}

Iterar un ArrayList utilizando Iterator


Iterator iterador = palabras.iterator();
while( iterador.hasNext() ){
   System.out.println( iterador.next() );
}
