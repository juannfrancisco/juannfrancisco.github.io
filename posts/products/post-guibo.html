Background

Programar aplicaciones web compatibles con múltiples dispositivos, adaptables a diferentes resoluciones, con interfaces interactivas, desarrollando componentes de interface reutilizables y utilizando diferentes servicios backend es el objetivo principal de este curso, aprendiendo a utilizar tecnologías como Bootstrap 4 en conjunto con el framework AngularJS.


Descripción del Programa

AngularJS es un framework de JavaScript de código abierto, mantenido por Google, que ayuda con la gestión de lo que se conoce como aplicaciones de una sola página. Su objetivo es aumentar las aplicaciones basadas en navegador con capacidad de Modelo Vista Controlador (MVC).

Bootstrap es un framework o conjunto de herramientas de software libre para diseño de sitios y aplicaciones web. Contiene plantillas de diseño con tipografía, formularios, botones, cuadros, menús de navegación y otros elementos de diseño basado en HTML y CSS.


Destinatarios

Personas con conocimientos básicos en desarrollo de aplicaciones web o de escritorio en cualquier lenguaje. Debe poseer conocimientos básicos de programación como estructuras de control y entendimiento de diagramas de flujo básicos.
