Prorgama para Calcular el promedio de notas

Este ejercicio, esta pensado para practicar la composicion de objetos.

Posted by Juan Francisco el 12 de Mayo, 2016

Ejercicio

Un Alumno tiene una lista de notas y un Nombre, Una nota tiene el valor de la nota y un porcetaje, con el cual el alumno podrá calcular su promedio.
(nota * porcentaje) / 100

  • Debe crear dos constructores uno que por defecto inicialice los atributos de cada clase por defecto.
  • El segundo constructor debera permitir ingresar cada atributo de la clase.
  • Usar getters y Setter para cada atributo de la clase.
  • La clase alumno, tendra un método que permitira calcular el promedio de notas.
